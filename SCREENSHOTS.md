# 📸 界面展示

## 启动应用

### 1. 运行启动脚本

**Windows:**
```
双击 start.bat
```

你会看到：
```
🚀 启动 Monel Gateway...
📦 编译项目...
✅ 编译成功
🌐 启动服务器...

访问地址:
  - 欢迎页: http://127.0.0.1:7890/
  - 管理后台: http://127.0.0.1:7890/admin.html

按 Ctrl+C 停止服务器
```

## 界面导航

### 页面 1: 欢迎页 (/)

**URL:** http://127.0.0.1:7890/

**特点：**
- 渐变紫色背景
- 项目名称和描述
- 三个按钮：
  - 管理后台（跳转到登录页）
  - 健康检查（打开 /health 端点）

### 页面 2: 登录页 (/admin.html)

**首次访问管理后台会看到登录页**

**布局：**
- 居中的白色卡片
- 标题："Monel Gateway - 管理后台"
- Auth Key 输入框（密码类型）
- 连接按钮（带登录图标）

**操作：**
1. 输入 Auth Key（默认: `your-global-secret`）
2. 点击"连接"按钮
3. 认证成功后进入仪表盘

**导航：** 点击左侧 "Provider 管理"

**功能：**

- Provider 列表表格

**表格列：**

**操作：**

- ✏️ 点击编辑图标：打开编辑对话框
- 🗑️ 点击删除图标：打开确认对话框

- 标题："添加 Provider"
- 4个输入框：
  - ID（必填）

  - Base URL（必填）
  - API Key（必填）
- 取消 / 保存 按钮

## 故障排除

### 问题：页面无法加载

**检查项：**
1. 服务器是否启动？
   ```
   查看终端是否显示：
   "gateway listening on http://127.0.0.1:7890"
   ```

2. 端口是否被占用？
   ```bash
   netstat -ano | findstr 7890
   ```

3. 浏览器控制台是否有错误？
   ```
   按 F12 打开开发者工具
   查看 Console 和 Network 标签
   ```

### 问题：登录失败

**检查项：**
1. Auth Key 是否正确？
   ```
   查看 config.yaml 中的 auth_key
   ```

2. 网络请求是否成功？
   ```
   打开浏览器开发者工具
   查看 Network 标签
   确认 /admin/config 请求返回 200
   ```

### 问题：保存配置失败

**检查项：**
1. YAML 格式是否正确？
2. 是否有写入权限？
3. 查看服务器日志获取详细错误
