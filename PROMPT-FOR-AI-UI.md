# Slint GUI 界面设计提示词

复制以下内容粘贴给 AI，让它生成 HTML 原型作为 Slint 实现的视觉参考。

---

```
为一个本地 API 网关桌面应用生成界面设计原型，单个 HTML 文件，仅展示布局和交互效果，不需要调用任何 API。深色主题，科技感。

功能页面：

1. 登录页 — 输入 Auth Key，点击连接

2. 仪表盘 — Server 状态（在线/离线）、监听地址、Provider 数量、模型总数、重新加载配置按钮

3. Provider 管理 — Provider 列表（ID/名称/Base URL/API Key），支持添加、编辑、删除，API Key 默认遮罩可切换显示，操作需确认对话框

4. 模型列表 — 按 Provider 分组，每组可折叠展开，显示模型名

5. 设置 — 编辑监听地址、端口、Auth Key

整体布局：左侧垂直导航栏 + 右侧内容区。用假数据填充。
```

---

用 AI 生成的 HTML 在浏览器中打开，作为 Slint 实现的视觉参考。
